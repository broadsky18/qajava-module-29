Обнаруженные ошибки:
1. Метод isTeenager() возвращает true для всех возрастов < 19, включая детей (например, 12 лет).
2. Метод isTeenager() возвращает false для 19 лет, хотя 19 — это тинейджер.

Причина ошибки:
- В коде используется условие `age < 19`, из-за чего 19 лет не считается тинейджерским возрастом.

Как найти ошибку:
- Тест для `isTeenager(12)` проваливается, потому что метод возвращает true (а должно быть false).
- Тест для `isTeenager(19)` проваливается, потому что метод возвращает false (а должно быть true).
